#W_Baxster

![W_Baxster](img)

A WiP handwired 65% keyboard with two rotary encoders and a speaker running on a Proton-C

[Layout](http://www.keyboard-layout-editor.com/#/gists/ff2ea05e9b76155214bb5b073e1253ba)

Keyboard Maintainer: [Condog9000](https://github.com/Condog9000)  
Hardware Supported: Proton-C 

Make example for this keyboard (after setting up your build environment):

    make condog9000/w_baxster:default
